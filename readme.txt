1) Про проект:
    Bitmap.h - класс Pixel
    Bitmap.cpp - функция вывода изображения в файл
    main.cpp - запуск сцены и заполнение изображения
    Tracing.h - описание классов примитивов - треугольник и сфера + описание класса сцены
    tracing.cpp - функция вычисления пересечения + функция освещения

2) Какие объекты постарался реализовать помимо базовой части:
    Использование текстур (+1).
    Использование SIMD инструкций (+2). Аргумент командной строки: -simd 1. - можно убедиться посмотрев асемблерный код 
    Использование многопоточности (+2). Аргумент командной строки: -threads 10. - использую OpenMp.
    Примитив бесконечной плоскости
